数项级数的基本概念

数项级数的概念

引入

  有限个实数相加一定是实数，那么无限个实数相加还是实数吗？为了解决这个问题，我们引入了级数的概念。

级数

  设$u_1,u_2,\ldots,u_n,\ldots$是一个给定的数列，按照数列{$u_n$}下标的大小依次相加，得到形式上和
$$u_1 + u_2 + \ldots + u_n +\ldots \tag 1$$
这个表达式称为数项级数,简称级数,记作$\sum_{n=1}^\infty{u_n}$.即：
$$\sum_{n=1}^\infty{u_n} = u_1 + u_2 + \ldots + u_n +\ldots$$
式中每一数称为级数的项，其中$u_n$称为级数（1）的通项（一般项）.

部分和

级数的前$n$项和
$$S_n = u_1 + u_2 + \ldots +u_n \tag 2$$
称为级数（1）的前n项部分和或者简称部分和。

收敛与发散

由部分和，我们得到了一个数列{$S_n$},且$\sum_{n=1}^{\infty}u_n = \lim_{n\to \infty}S_n$.若$\lim_{n\to \infty}S_n = S(常数)$,则称级数（1）收敛，并称$S$为级数的和。若$S$不存在，则称级数发散。

两个重要级数

等比级数（几何级数）

$$a + aq + aq^2 + \ldots + aq^{n-1} + \ldots = \sum_{n=1}^\infty aq^{n-1}$$

收敛性：

  1. 当$|q|<1$时，等比级数收敛，其和为$\frac{a}{1-q}$
  2. 当$|q|\geq 1$时，等比级数发散

$p$级数

$$1 + \frac{1}{2^p} + \frac{1}{3^n} + \ldots + \frac{1}{n^p} + \ldots = \sum_{n=1}^\infty\frac{1}{n^p}$$

收敛性：

  1. 若$p>1$，该级数收敛
  2. 若$q\leq 1$,该级数发散

数项级数的基本性质

性质一

$$若级数\sum_{n=1}^\infty u_n,\sum_{n=1}^\infty v_n都收敛，则这两个级数可以进行线性运算$$

性质二

  一个级数改变他的有限项，或者删除前面有限项，再或者在级数前面加上有限项，均不影响级数的收敛性

性质三

  若级数$\sum_{n=1}^\infty u_n$收敛，则在级数中任意添加括号所得到的新级数也收敛，且和不变。

性质四

  若级数$\sum_{n=1}^\infty u_n$收敛，则必有$\lim_{n\to \infty}u_n = 0$.

  重要运用:若$\lim_{n\to \infty }u_n$不存在或者其值不为0,则级数$\sum_{n=1}^\infty u_n$发散。
