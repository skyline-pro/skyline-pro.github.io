二极管的直流等效电路

发表于 2020-07-03 更新于 2021-02-24 分类于 电基

……

二极管的重要参数

  1. $I_F$:正向工作时的最大电流
  2. $U_R$:最高反向工作电压
  3. $I_R$:反向未击穿时的电流，越小反向截至效果越好
  4. $f_M$:二极管中电流的最高频率，如果过大，因为二极管中有电容效果，单向导电性就被破坏了。高频电路中需要考虑

二极管的等效电路

等效电路

二极管的微变等效

  利用大的直流电让二极管导通，然后在直流电压得位置附近体现小的交流电

$$r_d = \frac{U_T}{I_D}$$

$$i_d = \frac{U_i}{ r_d + R}$$

分析过程

  1. 分析直流，确定工作的$I_D$
  2. $r_d = \frac{U_T}{I_D}$
  3. 就可以用一个大小为$r_d$的电阻等效为二极管
  4. 一定是分析微小的交流电时才能这样分析

稳压二极管

一、特点

  工作于反向击穿状态下，电流变化大时电压基本不变，能通过较大的电流

二、伏安特性

  和普通二极管基本一样，只是反向击穿时能够工作。

  稳压二极管也是二极管，不一定就一直反向击穿了

三、主要参数

  • $\alpha$:温度变化一度，击穿电压变化多少度
  • $U_Z$:最低工作电压
